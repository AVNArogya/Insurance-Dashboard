⚠ Google Sheet Connection Failed
Follow one of these solutions to fix it
✅ Solution 1 — Check Google Sheet is Published (Most Common Fix)
  1. Open your Google Sheet
  2. Click File → Share → Publish to web
  3. Select your sheet tab → choose CSV
  4. Click Publish and confirm
  5. Copy the URL — paste it in the HTML file at SHEET_CSV_URL
  6. The URL must end with: output=csv
Correct URL format:
https://docs.google.com/spreadsheets/d/SHEET_ID/pub?gid=0&single=true&output=csv
🚀 Solution 2 — Open from a Web Server (Best for Office Use)

Browsers block Google Sheet fetch from local files. Host the HTML on a simple web server:

Option A — Python (if Python installed):
python -m http.server 8080 Then open: http://localhost:8080/AVN_Arogya_Live_Dashboard.html

Option B — VS Code Live Server extension:
Install "Live Server" → Right click HTML file → Open with Live Server

Option C — Upload to Google Drive → open with browser
📁 Solution 3 — Use Google Apps Script as a Proxy (No CORS issue)

Create a Web App from Google Sheets that serves your data without CORS:

  1. In Google Sheet: Extensions → Apps Script
  2. Paste this code:
function doGet(){
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  const data  = sheet.getDataRange().getValues();
  const headers = data[0];
  const rows = data.slice(1).map(r=>{
    const obj={};
    headers.forEach((h,i)=>obj[h]=r[i]);
    return obj;
  });
  return ContentService
    .createTextOutput(JSON.stringify(rows))
    .setMimeType(ContentService.MimeType.JSON);
}
  1. Click Deploy → New deployment → Web App
  2. Execute as: Me | Access: Anyone
  3. Copy the Web App URL
  4. In the dashboard HTML, change SHEET_CSV_URL to this URL
    and set USE_APPS_SCRIPT = true
